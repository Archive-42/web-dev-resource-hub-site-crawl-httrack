Become a sponsor to OpenSSL

openssl

OpenSSL is a robust, commercial-grade, and full-featured toolkit for the Transport Layer Security (TLS) and Secure Sockets Layer (SSL) protocols. It is also a general-purpose cryptography library.

The OpenSSL project relies on funding to maintain and improve OpenSSL.

We obtain funds from commercial support contracts and sponsorship donations. We use those funds to pay for full time fellows to work on maintaining and developing OpenSSL as well as infrastructure, meets, etc.

Please note that although we are very appreciative of donations, sponsorship does not give any additional rights or benefits. For example, we do not share information about security issues in advance with GitHub sponsors.
